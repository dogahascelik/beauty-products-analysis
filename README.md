# 🛍️ Beauty Products Data Analysis

This project explores a dataset of beauty products using Python and Pandas. The goal is to extract insights related to product pricing, customer ratings, and brand distribution in an e-commerce setting.

## 📁 Files
- `BeautyProducts.ipynb`: Main Jupyter Notebook with all analysis steps
- `beautyproducts_sample.csv`: Sample of the full dataset (first 1000 rows)

## 📊 Key Insights
- Distribution of product prices and ratings
- Correlation between price and rating
- Top-rated products by brand and category
- Common keywords in product titles

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib & Seaborn

- Creating insightful visualizations for e-commerce
